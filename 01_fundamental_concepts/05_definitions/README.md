## 1.5.1
The Fibonacci numbers `f0`,`f1`,`...` are defined by rule that `f0 = 0`, `f1 = 1` and `f(n+2) = f(n) + f(n+1)` for all `n >= 0`.

Give a definition of the function `fib` that takes an integer `n` and returns `f(n)`.
